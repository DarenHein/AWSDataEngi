## Certificacion AWS Data Engineer 

### puntos claves de la certificacion 

nombre : AWS Certified Data Engineer Associate 

** 1 Certificacion tecnica enfocada en el diseño , construccion y operacion de soluciones 
de datos en AWS 

** 2 Enfocado en ecenarios reales de ing de datos , ingestion , transformacion 
almacenamiento , procesamiento y analisis de datos 

** 3 uso de Servicios : 

	* Amazon S3 
	* AWs Glue 
	* Amazon Redshift 
	* Amazon Athena 
	* Amazon Kinesis 
	* AWS Lambda 
	* Step Functions 
	* EMR 

** 4 Evalua la capacidad para diseñar arquitecturas de datos escalables en 
AWS , incluyendo : 
	
	* Data Lakes 
	* pipelines 
	* ETL/ELT
	* procesamiento batch y streming 
	* seguridad 
	* governanza de datos
